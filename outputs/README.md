# outputs/

Ce dossier est rempli automatiquement lors de l'exécution des notebooks :

- `figures/` : courbes d'entraînement, matrices de confusion, visualisations de feature maps (.png), générées par les trois notebooks.
- `models/` : poids des modèles entraînés (.pth) pour chaque partie (MLP, CNN, Seq2Seq).

Ces fichiers ne sont pas pré-générés dans le repository (ils dépendent du runtime d'exécution et du GPU disponible) ; ils apparaissent après une première exécution complète des notebooks correspondants.
